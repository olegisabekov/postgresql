Система реализует универсальное хранилище телеметрии на базе реляционной БД с использованием EAV-модели и типизированных таблиц значений. Обеспечивает гибкое добавление новых параметров, поддержку различных типов данных и хранение исторических значений без изменения схемы базы данных.

The system implements a universal telemetry storage based on a relational database using an EAV model and typed value tables. It provides flexible addition of new parameters, supports multiple data types, and enables storage of historical values without requiring changes to the database schema.
